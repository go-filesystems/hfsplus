# hfsplus

[![License: BSD-3-Clause](https://img.shields.io/badge/License-BSD%203--Clause-blue.svg)](https://opensource.org/licenses/BSD-3-Clause)

A pure-Go driver for the **HFS+** (Mac OS Extended) on-disk format — part of the
[go-filesystems](https://github.com/go-filesystems) family.

> **Status: reserved / planned.** This repository is a placeholder; the HFS+
> driver is not yet implemented. The sibling repos (`apfs`, `ext4`, `xfs`,
> `btrfs`, `ntfs`, `zfs`, `exfat`, `fat32`, `iso9660`, `squashfs`, `ufs`,
> `ffs`, `uefi`) show the conventions a future implementation will follow:
> interface-first drivers over the shared
> [`interface`](https://github.com/go-filesystems/interface) contract, read +
> write support, `Mkfs`, and a native + emulated six-architecture CI with a
> coverage gate.
